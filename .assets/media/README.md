# `.assets/media/` directory
> **:open_file_folder: Parent directory for media assets.**

Add media files to the one of `.assets/media/`'s sub-directories based on the medium-type:

* **`audio/`**, e.g., `.ogg`, `.m4a`
* **`img/`**, Still images e.g., `.gif`, `.png`, and `.jpeg`
* **`video/`**, e.g., `.mov`
